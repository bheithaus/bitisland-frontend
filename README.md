coffeeboiler
============

# Express Angular CoffeeScript MongoDB Boilerplate
This is meant to be a simple, easy to use seed for a rich client side Application, similar to MEAN stack, but in Coffeescript.


## Technologies

- Express
- Angular
- CoffeeScript
- MongoDB
- Socket.io
- JWT token authentication
- Twitter Bootstrap 3

## Installation / Usage

in the terminal
`Git clone repo
npm install
bower install

gulp`

And you're off!
